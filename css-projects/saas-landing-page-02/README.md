# 🎓 Learn Academy Landing Page

A modern and responsive landing page for an online learning platform built using **HTML5** and **CSS3**. The project showcases semantic HTML, CSS Grid, Flexbox, responsive web design, interactive UI elements, and modern styling techniques without the use of JavaScript.

---

## 📋 Features

- Responsive landing page design
- Semantic HTML5 structure
- Sticky navigation bar with smooth scrolling
- Hero section with call-to-action buttons
- Course showcase using CSS Grid
- Featured course with highlighted badge
- Statistics section
- Student testimonials
- Contact form
- Interactive hover animations
- Custom CSS variables for consistent theming
- Mobile-friendly responsive layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- CSS Variables
- Flexbox
- CSS Grid
- Media Queries
- CSS Transitions
- Pseudo-elements (`::before`, `::after`)
- Responsive Web Design
